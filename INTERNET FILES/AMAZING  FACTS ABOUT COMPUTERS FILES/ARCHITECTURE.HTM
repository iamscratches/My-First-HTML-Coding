CLASSIFICATION OF COMPUTER NETWORKING BASED ON ARCHITECTURE

ARCHITECTURE THE ARCHITECTURE OF A NETWORK CAN BE DEFINED AS THE STRUCTURE & ORGANIZATION OF THE COMPUTER'S HARDWARE & SOFTWARE
NETWORKS CAN BE CLASSIFIED AS CLIENT/SERVER & PEER TO PEER(P2P)ARCHITECTURES

CLIENT/SERVER NETWORKING

IN CLIENT/SERVER NETWORKING THERE IS A CENTRALISED POWERFUL COMPUTER KNOWN AS THE SERVER WHICH DOES ALL THE PROCESSING. ALL THE PC'S OR NODES IN THE NETWORKS WHICH IS KNOWN AS CLIENTS ARE CONNECTED TO THE SERVER.THE SERVER MACHINE CONTROLS THE NETWORK.THESE NETWORKS ARE EXPENSIVE TO SET UP.

PEER TO PEER(P2P) NETWORKING

IN A PEER TO PEER(P2P) NETWORKING THERE IS NO SERVER MACHINE INSTEAD THERE IS A DIRECT CONTACT BETWEEN THE NODES. THIS ARCHITECTURE IS BETTER FOR SMALLER NETWORKS AND ARE IN EXPENSIVE TO SETUP & MAINTAIN.
