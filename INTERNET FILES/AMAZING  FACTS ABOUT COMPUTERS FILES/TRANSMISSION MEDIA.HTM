CLASSIFICATION OF COMPUTER NETWORKING BASED ON TRANSMISSION MEDIA

TRANSMISSION MEDIA TRANSMISSION MEDIA REFERS TO THE WAY THE DATA IS TRANSFERED TO ONE NODE TO ANOTHER NODE ON A NETWORK.THE COMPUTER IN A NETWORK CAN BE LINKED BY CABLES.THE THREE TYPES OF CABLES USED TO NETWORK COMPUTERS ARETWISTED PAIR CABLES(REGULAR PHONES LINES ),COAXIAL CABLES ,& FIBRE OPTICS CABLE.FOR LONGER DISTANCE WHERE CABLING IS NOT PRACTICALLY POSSIBLE RADIO WAVES,MICRO WAVES,OR SATTELITES ARE USED TO TRANSMIT DATA.
