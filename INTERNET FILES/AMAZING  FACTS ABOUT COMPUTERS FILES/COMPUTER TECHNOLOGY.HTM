--------------------------THE COMPUTER SYSTEM-----------------------------

THE COMPUTER IS AN ELECTRONIC DEVICE THAT CAN PERFORM VARIETY OF OPERATION ACCORDING TO A SET OF INSTRUCTION CALLED PROGRAMS. THE COMPUTER WORKS ON THE PRINCIPLE OF INPUT-PROCESSING-OUTPUT CYCLE.
IN THIS CYCLE THE COMPUTER SYSTEM:-
  • RECIEVES DATA AND INSTRUCTIONS FROM THE USER.
  • PROCESS THE DATA ON THE BASICS OF THE INSTRUCTIONS GIVEN.
  • DISPLAYS THE PROCESSED DATA AS OUTPUT.

    SO AS TOLD THE MAIN COMPONENTS OF THIS CYCLE ARE
    • INPUT UNIT.
    • PROCESSING UNIT.
    • OUTPUT UNIT.
